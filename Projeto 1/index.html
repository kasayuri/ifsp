Fogão Inclusivo — autonomia, segurança e design para todos

Apresento o Fogão Inclusivo: projetado para oferecer uma experiência acessível, simples e segura para pessoas com deficiência, idosos e toda a família. Tecnologia e empatia unidas para transformar a rotina na cozinha!


As cores, formas, sons, texturas ou textos foram estrategicamente pensados para atender as diversas necessidades dos usuários, garantindo uma experiência inclusiva e intuitiva.

Principais benefícios:

  • Interface tátil: O fogão possui botões táteis com texturas diferentes para facilitar a identificação das bocas.
  • Indicadores sonoros: O fogão emite sinais sonoros para confirmar ações, como ligar ou ajustar a temperatura.
  • Controles intuitivos: Os controles são organizados de forma lógica e fácil de entender, com ícones claros para cada função.
  • Feedback tátil: O fogão fornece feedback tátil ao toque dos botões, ajudando na confirmação da ação.
  • Autonomia: controles táteis e texturizados que facilitam a identificação, mesmo sem olhar.
  • Segurança: desligamento automático, alarmes sonoros e indicadores visuais de alta visibilidade.
  • Usabilidade: painel intuitivo com ícones claros e feedback tátil e sonoro.
  • Conectividade opcional: app complementar para ajustes remotos e alertas.
  • Design inclusivo: acabamento fácil de limpar e marcações em alto contraste.

Outras funcionalidades:

  • O painel de controle possui ícones visuais claros e display digital para facilitar a compreensão das funções.
  • Aplicativo complementar que permite o controle remoto do fogão, com notificações visuais para alertas importantes.
  • Cadastro de contato de emergência no aplicativo, para suporte rápido em caso de necessidade.
  • Desligamento automático após certo período de tempo, garantindo segurança adicional.
  • O cozimento por indução reduz o risco de queimaduras, tornando o fogão mais seguro para todos os usuários.
  • Recomendamos instalar em mesas com regulagem de altura através de motor elétrico para facilitar o acesso por pessoas em cadeiras de rodas, caso necessário.

Instruções para utilização

Guia de Uso Acessível do Fogão

Este guia tem como objetivo fornecer instruções claras e acessíveis para o uso da nova proposta de interface de fogão inclusivo.

Para garantir o uso seguro e eficiente do fogão, siga estas instruções:

  1. Certifique-se de que todas as partes estejam corretamente montadas.
  2. Identifique os botões táteis com texturas diferentes.
  3. Use os botões para ligar o fogão e ajustar a temperatura.
  4. Ouça os indicadores sonoros para confirmar suas ações.

Descrição dos botões:

Botão Função
Liga/Desliga Liga ou desliga o fogão
Botão de seta para cima Aumenta a temperatura da boca do fogão mais próxima
Botão de seta para baixo Diminui a temperatura da boca do fogão mais próxima
Botão + Aumenta o volume dos sinais sonoros
Botão - Diminui o volume dos sinais sonoros
Timer Define o tempo de cozimento
